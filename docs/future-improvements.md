# Sentry Future Improvements Roadmap

The following are the planned next steps to take Sentry from a high-fidelity frontend portal to an active production deployment:

## 1. Real-Time Emergency Alerts
*   **WebSockets**: Implement a Socket.io server to push new camera detections immediately to the police overview screen without page reloads.
*   **Audio Alerts**: Trigger discrete audio alarms in command centers for critical (Level 5) severity incidents.

## 2. Integrated Camera Streams
*   **RTSP Playback**: Replace player placeholders with active video streams (e.g. using HLS or Dash players).
*   **Video Snapshots**: Fetch vehicle crop bounding boxes generated by the YOLOv8 model directly onto the claims and police report detail cards.

## 3. Advanced Dispatch Telemetry
*   **Google Maps API**: Render a live incident map displaying GPS hotspots and nearest police/ambulance dispatch vehicle coordinates.
*   **ETA Calculations**: Automate response team route navigation to estimate arrival times.

## 4. Mobile Apps for Field Officers
*   **Push Notifications**: Push dispatch notifications directly to mobile units.
*   **Evidence Collection**: Enable field officers to upload site photographs and witness statements directly into the Sentry incident log.
